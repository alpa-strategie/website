# Project Brief - Alpa Stratégie Website

## Project Identity
- **Name**: Alpa Stratégie – Baptiste Leroux
- **Type**: Personal Consulting / Portfolio Website
- **Repository**: git@github.com:alpa-strategie/website.git
- **Primary Language**: TypeScript/React
- **Framework**: Next.js with App Router

## Core Purpose
Build a professional, bilingual (French/English) personal website for Baptiste Leroux, founder of Alpa Stratégie, to:
- Present his expertise in digital transformation, IT governance, and performance management,
- Showcase key missions, case studies, and achievements,
- Reinforce personal credibility and visibility for interim management or consulting roles (PMO / CTO / CPTO),
- Generate qualified contacts from recruiters, executives, and consulting firms.

## Key Requirements

### Functional Requirements
1. **Multi-language Support**: (FR / EN) with a lightweight translation system.
2. **Core Pages**: 
    - Home – positioning, headline, and call-to-action
    - Profile/About – biography, philosophy, credentials
    - Expertise / Services – PMO, CTO, Digital Factory, Transformation
    - Missions / Case Studies – selected project summaries
    - Contact – simple lead form or mailto CTA
    - Landing pages - simple lead form with CTA
3. **Responsive Design**: mobile-first layout, optimized typography and spacing.
4. **Type Safety**: Full TypeScript implementation
5. **Modern Architecture**: Next.js App Router conventions

### Technical Requirements
1. **Framework**: Next.js (latest) with App Router
2. **Styling**: Tailwind CSS + custom brand palette (Alpa Stratégie blue / neutral gray / white)
3. **i18n**: Minimal custom context or next-intl / i18next integration
4. **Type Safety**: enforced with TypeScript
5. **Build Tools**: ESLint + Prettier configuration

### Non-Functional Requirements
1. **Performance**: fast load, optimized images, lazy loading where possible.
2. **SEO**: meta tags, schema, and Open Graph for LinkedIn sharing.
3. **Maintainability**: modular components, reusable layouts, clear folder structure.
4. **Accessibility**: WCAG-aligned semantic markup, keyboard navigation.

## Project Scope

### In Scope
- Bilingual website (FR / EN)
- Responsive navigation + footer
- Key sections: Home, Profile, Expertise, Missions, Contact
- SEO-ready meta structure
- Basic animation and motion design for premium look

### Out of Scope (Current Phase)
- Blog / News section
- CMS integration (Notion, Sanity, etc.)
- E-commerce or payments
- Authentication or admin dashboard

## Success Criteria
1. Clear positioning for Digital Strategy & IT Governance Consulting
2. Fully responsive and bilingual
3. Optimized SEO + share-friendly pages (LinkedIn cards)
4. Clean, maintainable codebase (Next.js App Router + Tailwind + TypeScript)
5. Deployed and live at alpa-strategie.com, ready for recruiter and partner traffic

## Project Constraints
- Must highlight Baptiste Leroux as the core brand persona
- Must follow Next.js App Router best practices
- Must support both French and English
- Must remain lightweight and easily editable (no CMS dependency)
- Must align visually with Alpa Stratégie brand (neural-network A logo + modern minimalist style)

## Suggested Content Tone & Visual Direction
- **Tone**: expert, clear, modern, credible, bilingual executive presence.
- **Visuals**: subtle motion, white + gray backgrounds, accent color from logo, strong typography hierarchy.
- **CTA**: “Let’s Talk” / “Contact Me” leading to email or calendar.
